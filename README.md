# Rifa Fresita Linda

Página lista para publicarse en GitHub Pages.

## Publicar en GitHub

1. Inicia sesión con el usuario `rivas0MC`.
2. Crea un repositorio nuevo llamado `rifa-fresita-linda`.
3. Sube el archivo `index.html`.
4. En el repositorio entra a **Settings → Pages**.
5. En **Build and deployment**, elige **Deploy from a branch**.
6. Selecciona la rama `main` y la carpeta `/ (root)`.
7. Guarda los cambios.

La dirección quedará aproximadamente así:

https://rivas0MC.github.io/rifa-fresita-linda/

## Marcar números vendidos

Abre `index.html` y busca esta línea:

```js
const numerosVendidos = [];
```

Agrega los números vendidos separados por comas. Ejemplo:

```js
const numerosVendidos = [2, 7, 15, 26, 40];
```

Guarda el archivo y vuelve a subirlo a GitHub.
